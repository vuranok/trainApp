* Command for building the project:
mvn clean package

* Command for running the project:
java -jar energy-consumption-0.0.1-SNAPSHOT.jar

* Bonus task implemented with "org.thymeleaf". 
In order to upload profile(s) and meter reading(s) csv files open your browser and type the following:
http://localhost:8080/

* REST APIs:
An example SoapUI project for REST resources are included in the "EnergyConsumption-soapui-project.xml" file under the project's root folder.
